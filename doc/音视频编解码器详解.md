# 音视频编解码器详解

## 一、基本概念

### 1. 标准 vs 编解码器

- **编码标准**（如 H.264、H.265）：由组织制定的规范，定义编码/解码算法。
- **编解码器（Codec）**：实现该标准的软件或硬件。
- 日常说的「H.264 编解码器」通常指：遵循 H.264 标准的编解码实现。

### 2. 硬编解码 vs 软编解码

| 类型 | 实现方式 | 典型代表 |
|------|----------|----------|
| **软编解码** | 用 CPU 执行算法 | FFmpeg、x264、libopus |
| **硬编解码** | 用 GPU、DSP、专用芯片执行 | NVENC、MediaCodec、VideoToolbox |

---

## 二、硬编解码器 vs 软编解码器 对比

| 维度 | 软编解码 | 硬编解码 |
|------|----------|----------|
| **执行单元** | CPU | GPU、DSP、专用芯片 |
| **功耗** | 高（CPU 满载） | 低（专用电路） |
| **发热** | 明显 | 相对较小 |
| **速度** | 较慢，依赖 CPU 性能 | 快，有专用加速 |
| **画质/音质** | 可调参数多，质量更好 | 参数少，质量略逊 |
| **兼容性** | 好，纯软件实现 | 依赖设备是否支持 |
| **灵活性** | 高，可升级算法 | 低，硬件固定 |
| **CPU 占用** | 高 | 基本不占 |
| **成本** | 无额外硬件成本 | 需要支持硬编解码的芯片 |

**适用场景**：
- **软编**：专业剪辑、高质量转码、老旧设备、兼容性优先
- **硬编**：直播、实时通话、手机录像、4K 实时编码、省电优先

## 三、视频编解码器

### 1. 视频软编码器

| 名称 | 格式 | 说明 |
|------|------|------|
| **x264** | H.264 | 开源，质量高，FFmpeg 常用 |
| **OpenH264** | H.264 | Cisco 开源，WebRTC 常用 |
| **x265** | H.265 | 开源，高压缩率 |
| **libvpx** | VP8/VP9 | Google 开源 |
| **libaom** | AV1 | AOMedia 开源 |
| **SVT-AV1** | AV1 | Intel 开源，编码较快 |
| **JM / HM** | H.264/H.265 | 官方参考实现，主要用于研究 |

| **Video Encode Engine (VCE)** | AMD | H.264、H.265 |
| **MediaCodec** | Android | H.264、H.265、VP8、VP9 |
| **VideoToolbox** | Apple | H.264、H.265、ProRes |

### 4. 视频硬解码器

## 五、H.264 与 H.265 详解

### 1. 是什么

| 标准 | 全称 | 发布 | 特点 |
|------|------|------|------|
| **H.264** | Advanced Video Coding | 2003 | 兼容性最好，设备支持最广 |
| **H.265** | High Efficiency Video Coding | 2013 | 同画质下码率约为 H.264 一半 |

### 2. 对比

| 项目 | H.264 | H.265 |
|------|-------|-------|
| 压缩效率 | 基准 | 约 2 倍 |
| 编码速度 | 快 | 较慢 |
| 解码负担 | 低 | 较高 |
| 兼容性 | 很好 | 一般 |

### 3. 使用示例（FFmpeg）

```bash
# H.264 编码
ffmpeg -i input.mp4 -c:v libx264 -crf 23 -preset medium output.mp4

# H.265 编码
ffmpeg -i input.mp4 -c:v libx265 -crf 28 -preset medium output.mp4
```

### 4. 选择建议

| 场景 | 建议 |
|------|------|
| 通用、兼容性 | H.264 |
| 4K、省带宽 | H.265 |
| 低端设备 | H.264 |
| 直播、会议 | H.264 |

## 六、常见编解码库/框架

| 库/框架 | 用途 |
|---------|------|
| **FFmpeg** | 音视频编解码、转码、封装 |

## 七、按场景选编解码器

| 场景 | 视频 | 音频 |
|------|------|------|
| 通用播放 | FFmpeg | FFmpeg |
| 实时通话 | OpenH264、VP8、硬件 H.264 | Opus、G.711 |
| 流媒体 | x264、x265、硬件编码 | FDK-AAC、Opus |
| 移动端 | MediaCodec、VideoToolbox | 系统音频 API |
| Web | VP9、AV1、H.264 | Opus、AAC |
