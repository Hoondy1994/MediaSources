# 音视频压缩算法详解

## 一、视频压缩算法

### 1. 基于块的混合编码（主流）

| 算法/标准 | 组织 | 特点 |
|-----------|------|------|
| **H.264/AVC** | ITU-T + MPEG | 块大小 4×4～16×16，运动估计/补偿，帧内/帧间预测，CABAC/CAVLC 熵编码 |
| **H.265/HEVC** | ITU-T + MPEG | 更大 CTU（最大 64×64），更多预测模式，压缩率约为 H.264 的 2 倍 |
| **H.266/VVC** | ITU-T + MPEG | 更复杂划分、预测和变换，压缩率再提升约 50% |
| **VP8** | Google | 8×8 块，简单运动估计，开源 |
| **VP9** | Google | 更大超级块（64×64），多参考帧，开源 |
| **AV1** | AOMedia | 更灵活划分、预测、变换，开源，压缩率优于 VP9 |
| **MPEG-2** | MPEG | 16×16 宏块，DVD、数字电视常用 |
| **MPEG-4 Part 2** | MPEG | 可变块大小，早期网络视频常用 |

### 2. 常见技术手段

| 技术 | 作用 |
|------|------|
| **运动估计与补偿（ME/MC）** | 利用帧间相似性，只编码运动矢量与残差 |
| **帧内预测** | 利用空间相邻块预测当前块 |
| **变换编码（DCT/DST）** | 将残差变换到频域，便于压缩 |
| **量化** | 降低系数精度，主要损失来源 |
| **熵编码（Huffman、CABAC、CAVLC）** | 对系数做无损压缩 |
| **环路滤波（去块、SAO）** | 减轻块效应，提升主观质量 |

### 3. 其他思路

| 类型 | 代表 | 说明 |
|------|------|------|
| **帧内编码** | JPEG、MJPEG | 每帧独立编码，无帧间预测 |
| **帧差编码** | 简单帧差 | 只编码帧间差异 |
| **小波编码** | JPEG 2000、VC-2 | 小波变换替代 DCT |
| **神经网络** | 各类 AI 编码器 | 用深度学习做预测和压缩 |

## 二、音频压缩算法

### 1. 有损压缩

| 算法/标准 | 组织 | 主要技术 |
|-----------|------|----------|
| **MP3** | MPEG | 心理声学模型、子带编码、MDCT、量化、Huffman |
| **AAC** | MPEG | 改进 MDCT、TNS、预测、更多声道模式 |
| **Opus** | IETF | SILK（语音）+ CELT（音乐）混合，低延迟 |
| **Vorbis** | Xiph | 改进 MDCT、矢量量化、码本 |
| **WMA** | 微软 | 类似 MDCT + 心理声学 |
| **AC3/Dolby Digital** | Dolby | 变换编码、声道耦合、心理声学 |

### 2. 无损压缩

| 算法 | 主要技术 |
|------|----------|
| **FLAC** | 线性预测 + 残差 Rice 编码 |
| **ALAC** | 线性预测 + 熵编码 |
| **APE** | 预测 + 范围编码 |
| **WAV** | 无压缩，仅 PCM 存储 |

### 3. 语音专用

| 算法 | 类型 | 说明 |
|------|------|------|
| **G.711** | 波形编码 | A-law/μ-law，64 kbps |
| **G.722** | 子带 ADPCM | 宽频语音，64 kbps |
| **G.729** | 参数编码 | CS-ACELP，8 kbps |
| **AMR** | 自适应多速率 | 3G 语音 |
| **SILK** | 混合编码 | Opus 语音部分 |
| **EVS** | 增强语音 | 4G/5G 语音 |

### 4. 心理声学相关

| 技术 | 作用 |
|------|------|
| **掩蔽效应** | 强音掩蔽弱音，可丢弃被掩蔽成分 |
| **临界频带** | 按人耳感知划分频带 |
| **时域掩蔽** | 前后掩蔽，用于时域处理 |
| **频域掩蔽** | 同时掩蔽，用于频域量化 |

## 三、压缩算法对比（简要）

| 维度 | 视频 | 音频 |
|------|------|------|
| **空间冗余** | 帧内预测、变换 | 声道间冗余、预测 |
| **时间冗余** | 运动估计与补偿 | 帧间预测 |
| **统计冗余** | 熵编码 | 熵编码 |
| **感知冗余** | 量化、滤波 | 心理声学掩蔽 |
| **编码复杂度** | 高（搜索、变换） | 中（变换、心理声学） |
